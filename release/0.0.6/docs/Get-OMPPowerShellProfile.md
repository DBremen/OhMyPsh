# Get-OMPPowerShellProfile

## SYNOPSIS
Retrieves the current session PowerShell profile.

## SYNTAX

```
Get-OMPPowerShellProfile [[-ProfileType] <String>]
```

## DESCRIPTION
Retrieves the current session PowerShell profile.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-OMPPowerShellProfile
```

## PARAMETERS
