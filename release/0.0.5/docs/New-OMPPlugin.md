# New-OMPPlugin

## SYNOPSIS
Creates a new OMP Plugin template.

## SYNTAX

```
New-OMPPlugin [-Name] <String>
```

## DESCRIPTION
Creates a new OMP Plugin template.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
New-OMPPlugin -Name 'mygreatplugin'
```

Creates 'mygreatplugin' in the plugins directory and displays the full path to the created plugin.

## PARAMETERS

### -Name
Name of the plugin
