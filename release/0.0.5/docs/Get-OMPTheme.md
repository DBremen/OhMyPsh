# Get-OMPTheme

## SYNOPSIS
Shows themes and their load state.

## SYNTAX

```
Get-OMPTheme [[-Name] <String>]
```

## DESCRIPTION
Shows themes and their load state.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-OMPTheme
```

Shows all OhMyPsh themes and if it is loaded or not.

## PARAMETERS

### -Name
The theme name.
If nothing is passed all themes are listed.
