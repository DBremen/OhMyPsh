# Get-OMPPowerShellProfileState

## SYNOPSIS
Retrieves all Powershell profiles for the exisisting session, if they exist, and the order of operation they are processed.

## SYNTAX

```
Get-OMPPowerShellProfileState
```

## DESCRIPTION
Retrieves all Powershell profiles for the exisisting session, if they exist, and the order of operation they are processed.
