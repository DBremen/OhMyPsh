# Optimize-OMPProfile

## SYNOPSIS
Runs ngen on powershell assemblies.
This can sometimes optimize startup times for PowerShell.

## SYNTAX

```
Optimize-OMPProfile
```

## DESCRIPTION
Runs ngen on powershell assemblies.
This can sometimes optimize startup times for PowerShell.
Requires an elevated prompt to run.
