# Remove-OMPPlugin

## SYNOPSIS
Removes a loaded plugin

## SYNTAX

```
Remove-OMPPlugin [-Force] [-NoProfileUpdate]
```

## DESCRIPTION
Removes a loaded plugin

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Remove-OMPPlugin -Name o365
```

## PARAMETERS
